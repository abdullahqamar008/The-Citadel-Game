# 🏰 The Citadel - Text-Based Adventure Game (C++)

A simple yet interactive text-based adventure game built in C++.

---

## 🎮 Gameplay

* Start with 100 health
* 5 moves limit
* Choose directions (N/S/E/W)
* Different paths = different outcomes
* Collect a torch to unlock winning path

---

## 🧠 Concepts Used

* While loop
* Switch statements
* If-else conditions
* Variables and state management

---

## 💡 What I Learned

This project helped me understand how to build logic using basic programming concepts and turn them into a complete working program.

---

## ▶️ How to Run

g++ main.cpp -o game
./game
