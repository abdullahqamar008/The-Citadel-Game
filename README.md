The Citadel Text Based Adventure Game (C++)

Gameplay

* Start with 100 health
* 5 moves limit
* Choose directions (N/S/E/W)
* Different paths = different outcomes
* Collect a torch to unlock winning path

Concepts Used

* While loop
* Switch statements
* If-else conditions
* Variables and state management

How to Run

g++ main.cpp -o game
./game
